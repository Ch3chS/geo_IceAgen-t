# Directorio data/

## Descripción

Contiene los datos a utilizar, ya sea crudos, procesados o de fuentes externas.

## Estructura del directorio

- `external/` Datos de fuentes externas (INE, Copernicus, etc).
- `processed/` Datos ya procesados y limpios.
- `raw/` Datos originales sin procesar (o "crudos").
